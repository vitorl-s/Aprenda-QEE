# Aprenda QEE

Para executar o programa, abra o projeto no eclipse entre no 
pacote view e abra a classe Main.java, clique nela com o 
botão direito e selecione "Run as" e "Java aplication". 

## Menu Inicial

A tela do menu é a primeira a aparecer ao iniciar a execução do programa, nela
o usuário poderá escolher qual das duas simulações possíveis deseja fazer pelos botões que levam
para as proximas telas. 

## Simular Fluxo de Potência Fundamental

Após escolher esta simulação o usuário deverá preencher os dados da amplitude e 
angulo da tensão e da corrente e, em seguida, apertar o botão para simular a Onda respectiva. 

## Simular Distorção Harmonica

Ao escolher esta simulação o usuário deverá colocar os valores de amplitude e angulo
da componente fundamental e apertar o botão "Gerar Onda" em seguida, deve-se escolher a quantidade de harmônicos, definir a paridade
e em seguida apertar o botão "OK", com isso, surgirão os gráficos em branco ao lado de cada menu e deve-se preencher conforme o pedido por cada gráfico e após clicar no botão "Gerar" para cada gráfico desejado. 

## Observações:

Os valores da amplitude da corrente na simulação do fluxo de potência fundamental
deverá ser entre 0 e 100. Todas as outras amplitudes devem estar entre 0 e 220. Os angulos devem ser entre -180° e + 180°.  
Na Simulação de Distorção Harmônica o gráfico Resultante não foi Implementado, portanto a simulação para no gráfico de cada harmônico.
